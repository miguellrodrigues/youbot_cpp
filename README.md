# YouBot - Cpp

## Starting

This project contains the source code referring to the research of the student Miguel L. Rodrigues, member of the PET
group of Mechatronic Engineering at Cefet MG

### Pre requisites

C++ and WeBots

``` 
  Use the dll for your operating system
```

### Installing

Clone the repo

<ul>

  <li>mkdir build</li>
  <li>cd build</li>
  <li>cmake ..</li>
  <li>make</li>
  <li>./youbot_cpp</li>

</ul>
